Computer History

1970

  • Alan Shugart and a team of engineers invent the floppy disk, allowing data to be shared across computers
  • Many personal computers start to show up in stores including the Scelbi and Mark-8 Altair, IBM 5100, Radio Shack's TRS-80 and Commodore PET
  • Paul Allen and Bill Gates offered to write software for the Altair using the new basic language and form their own company called Microsoft
  • Steve Jobs and Steve Wozniak start apple computers and releases the Apple 1
  • Apple shows off the Apple 2. It offers colour graphics and incorporates audio casette drives for storage

1980

  • First IBM Personal computer released code-named "Acorn" and was the first computer you could obtain from outside distributors
  • Apple's Lisa was the first portable computer with a GUI, Drop Down Menu, and Icons
  • Microsoft releases windows, which was the company response to apple GUI, and Commodore unveils the Amiga 1000 which has advance audio and video fearures
  • On March 15 the first dot com domain was made, years before the World Wide Web would make internet history
  • Compaq releases the Deskpro 386 to stores. Its 32 bit architecture provide speed comparable to a mainframe

1990

  • Tim Berners-Lee develops HyperText Markup Language(HTML)
  • PCs become gaming machines as many games start to show up in stores
  • Sergey Brin and Larry Page make the google search engine
  • Microsoft invests 150 million to apple ending a court case against Microsoft for allegedly copying the "look and feel" of apple
  • Wi-fi becomes part of computing language and user are able to connect to the internet without a wire

2000

  • Apple releases the Mas OS X operating system then Microsoft releaes the windows Xp operating system
  • The first 64 bit processor, AMD Athalon 64, get released in stores
  • Mozila Firefox launches as competition for windows internet explorer for the dominant web browser.
  • Youtube and Facebook launch
  • Apple released the MacBook Pro, the first intel based, dual core, mobile computer and the intel based iMac

2010

  • Apple unveils the iPad changing the way people view media
  • Google releases the Chromebook a laptop running the Google Chrome OS
  • Facebook Gains 1 billion users on October 4
  • Apple watch is released and Windows 10 releases
  • The first reprogrammable quantum computer is was created
